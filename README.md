# discordpuppets
Control discord webhook puppet bots

#### Requirements

[Yarn](https://yarnpkg.com/en/)

#### Run dev server

```
yarn run dev
```

#### Create production build

```
yarn run build
```

Place build `./dist` files on a webserver and navigate to the page. No other server side scipts like php required.
